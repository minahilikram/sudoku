ADA Sudoku
=======================


************
Compilation
************
    - open terminal

    RUN:
        - type "make"


********************
Running the program
********************
    RUN:
        - type "make run"


********************
Cleaning the files
********************
    RUN:
        - type "make clean"


*****************
Known Limitations
*****************
	- Does not solve the puzzle completely
	- Does not ask user for input of the puzzle
	- Does not output solution to file
